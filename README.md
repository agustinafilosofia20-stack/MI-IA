# MI-IA

Proyecto de **IA simulada** 
Simula una inteligencia artificial básica para interactuar con el usuario usando Node.js y frontend en HTML/JS.

## Archivos principales

- `index.html` – Interfaz del usuario.  
- `index.js` – Lógica de la IA simulada.  
- `package.json` y `package-lock.json` – Configuración y dependencias de Node.js.  
- `.env.example` – Variables de entorno de ejemplo.  
- `.gitignore` – Archivos ignorados por Git.  

## Uso

1. Clonar el repositorio:
```bash
git clone https://github.com/agustinafilosofia20-stack/MI-IA.git
Instalar dependencias:
npm install
Abrir index.html en el navegador para probar la IA.
